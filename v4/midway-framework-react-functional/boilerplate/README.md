# my_midway_project

## QuickStart

### Development

```bash
$ npm i
$ npm run dev
$ open http://localhost:5173/
```

### Build

```bash
$ npm run build
$ npm start
```

### Notes

- Midway server runs under `src/server` and exposes `/api/*`.
- React app runs with Vite and calls typed APIs from `src/server/api`.
- Use `npm run lint` to check code style.

[midway]: https://midwayjs.org
